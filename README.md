# Panel Web de Proyectos

Este proyecto es una aplicación Django para gestionar proyectos de desarrollo de software.

## Instalar entorno virtual
virtualenv -p "path/Pyton312/python.exe" env
cd .\env\
cd .\Scripts\
.\activate

## Instalación

1. Clona el repositorio.
2. Crea un entorno virtual.
3. Instala las dependencias: `pip install django`
4. Crear migraciones: `python manage.py makemigrations`
5. Ejecitar migraciones: `python manage.py migrate`
6. Corre el servidor: `python manage.py runserver`

## Tecnologías

- Python 3
- Django 4
- Bootstrap 5
- SQLite (por defecto)
